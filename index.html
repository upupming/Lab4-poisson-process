连续时间马尔科夫链

定义

随机过程 {X(t),t≥0}\{X(t), t \ge 0\}{X(t),t≥0} 的状态空间为 I={i0,i1,...,in,...∣n≥0}I = \{i_0, i_1, ..., i_n, ... | n \ge 0\}I={i0​,i1​,...,in​,...∣n≥0}，若对任意的 0<t1<t2<...<tn<...0 < t_1 < t_2 < ... < t_n < ...0<t1​<t2​<...<tn​<...，有

(1)P{X(tn+1)=in+1∣X(t1)=i1,X(t2)=i2,...,X(tn)=in}=P{X(tn+1)=in+1∣X(tn)=in}\begin{aligned} & P\{X_{(t_{n+1})} = i_{n+1}|X_{(t_{1})} = i_{1}, X_{(t_{2})} = i_{2}, ..., X_{(t_{n})} = i_{n}\} \\ = & P\{X_{(t_{n+1})} = i_{n+1}|X_{(t_{n})} = i_{n}\} \end{aligned} \tag{1} =​P{X(tn+1​)​=in+1​∣X(t1​)​=i1​,X(t2​)​=i2​,...,X(tn​)​=in​}P{X(tn+1​)​=in+1​∣X(tn​)​=in​}​(1)

直观上理解为：在 tn+1t_{n+1}tn+1​ 时刻的状态只受 tnt_{n}tn​ 时刻状态的影响。

与离散时间马尔科夫链相比，多了一个时间的概念，之前用 X(n)X(n)X(n) 表示随机过程，其中的 nnn 是离散的；而这里 X(t)X(t)X(t) 表示随机过程，其中的 ttt 是连续的。

注意：在连续时间马尔科夫链中，时间是连续的，状态是离散的。

转移概率

在 sss 时刻处于状态 iii，经过时间 ttt 后转移到状态 jjj 的概率为：

(2)pij(s,t)=P{X(s+t)=j∣X(s)=i}p_{ij}(s, t) = P\{X(s+t) = j|X(s) = i\} \tag{2} pij​(s,t)=P{X(s+t)=j∣X(s)=i}(2)

定义齐次连续时间马尔科夫链，其『齐次转移概率』满足：

(2)pij(s,t)=pij(t)p_{ij}(s, t) = p_{ij}(t) \tag{2} pij​(s,t)=pij​(t)(2)

也就是说不管起始时间 sss 是多少，转移概率只与时间间隔 ttt 有关。

在后文中，我们只考虑齐次连续时间马尔科夫链。

泊松过程

定义 1

计数过程 {N(t),t≥0}\{N(t), t \ge 0 \}{N(t),t≥0} 称为具有参数 λ(λ>0)\lambda(\lambda > 0)λ(λ>0) 的泊松过程，如果：

  1. N(0)=0N(0) = 0N(0)=0
  2. 过程有独立增量
  3. 在任一长度为 ttt 的区间中事件的个数服从均值为 λt\lambda tλt 的泊松分布。即对一切 s,t≥0s, t \ge 0s,t≥0

    P(N(t)=n)=e−λtλtnn!,n=0,1,...P(N(t) = n) = e^{-\lambda t}\frac{\lambda t^n}{n!}, n = 0, 1, ... P(N(t)=n)=e−λtn!λtn​,n=0,1,...

定义 2

计数过程 {N(t),t≥0}\{N(t), t \ge 0 \}{N(t),t≥0} 称为具有参数 λ(λ>0)\lambda(\lambda > 0)λ(λ>0) 的泊松过程，如果：

  1. N(0)=0N(0) = 0N(0)=0
  2. 过程有平稳与独立增量
  3. 在 Δt\Delta tΔt 充分小时，在 (t,t+Δt)(t, t+\Delta t)(t,t+Δt) 内事件发生一次的概率为 λΔt+o(Δt)\lambda \Delta t + o(\Delta t)λΔt+o(Δt)，即 P(N(t+Δt)−N(t)=1)=λΔt+o(Δt)P(N(t+\Delta t)-N(t) = 1) = \lambda \Delta t+ o(\Delta t)P(N(t+Δt)−N(t)=1)=λΔt+o(Δt)
  4. 在 Δt\Delta tΔt 充分小时，在 (t,t+Δt)(t, t+\Delta t)(t,t+Δt) 内事件发生两次或以上的概率为 o(Δt)o(\Delta t)o(Δt)，即 P(N(t+Δt)−N(t)≥2)=o(Δt)P(N(t+\Delta t)-N(t) \ge 2) = o(\Delta t)P(N(t+Δt)−N(t)≥2)=o(Δt)
